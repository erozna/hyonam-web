System Overview

DOKKEBI
OS

DOKKEBI OS는 단순한 자동화 툴이 아닙니다.
수익을 연료로 삼아 물리적·지능적 신체를
스스로 키워나가는 인공 생태계입니다.

대장님의 추상적 명령 한 마디가
전략 수립 → 자원 배분 → 자본 창출로
자동 완결되는 시스템을 목표로 합니다.

01
대장님 이익 최우선 단기 효율보다 장기 성장 우선. 필요 시 즉각 능동적 자원 제안.
02
단일 진실 원칙 SHARED_BRAIN.json 단일 파일만 신뢰. SHA-256 무결성 보장.
03
비가역적 행동 통제 결제·계정 생성은 반드시 롤백 스냅샷 생성 후 인간 승인 必.
04
민주적 합의 다중 AI 토론을 통한 75% 동의 확보 시에만 의사결정 승인.
05
자원 절대 보호 NAS 기존 서비스 보호. 리소스 초과 시 즉시 작업 일시중지.
06
점진적 자기진화 전면 재작성 금지. 샌드박스 소규모 테스트 후 단계적 배포.
